# <h1 align="center">COFFEE-SALES_DATA-ANALYSIS ˙✧˖°☕ ༘ ⋆｡˚</h1>
Coffee Sales Data Analysis project that explored sales trends, customer behavior, and product performance using data visualization and analytical techniques. It provided insights into revenue patterns, top-selling products, and seasonal demand to support data-driven decision-making.

### <h4 align="center">🎯GOALS AND OBJECTIVES </h4>
- To analyze coffee sales performance across different time periods
- To identify top-selling products and low-performing items
- To understand customer purchasing behavior and trends
- To evaluate sales distribution by day, year, and time of day
